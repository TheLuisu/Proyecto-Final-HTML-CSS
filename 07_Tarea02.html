Tarea 02 - Lenguajes Informáticos

Un lenguaje informático es el usado por ordenadores,para transmitir información. Podemos encontrar múltiples como pueden ser : lenguaje de programación, lenguaje de especificación, lenguaje de consulta, lenguaje de marcas, lenguaje de transformación, protocolo de comunicaciones, lenguaje de sonido,lenguaje gráfico,pseudocódigo, etc.

Los lenguajes informáticos son considerados erróneamente como lenguajes de programación por muchas personas.
Los lenguajes informáticos se pueden clasificar en varias clases, entre las que se incluyen las siguientes:

Lenguaje informático Ejemplo
Lenguaje de programación Java - JavaScrip - Go - Kotlin - Python - C/C++ - C# - Ruby
Lenguaje de especificación Alloy - Autómatas - B - Cálculo Pi - CCS - CSP - Estelle - Larch
Lenguaje de consulta SQL - Common Query Languaje (CQL) - CODASYL - D - DMX - Datalog
Lenguaje de marcas HTML - XHTML - RDF - Wireless ML - S1000D - Rich Tex Format - ASN.1
Lenguaje de transformación Java/EMF - ATL - QVTr - QVTo - Tefkat - XSLT
Protocolo de comunicaciones TCP - IP - POP - SMTP - HTTP
Lenguaje de sonido Utilizados para crear sonidos
Lenguaje gráfico Utilizados para crear graficos, figuras, dibujos, etc.
Pseudocódigo Es una descripción de alto nivel compacta e informal del principio operativo de un programa informático u otro algoritmo.
Lenguaje de Definición de Procesos XPDL(XML Procces Definition Languaje) - WS-BPEL (Web Services Business Process Execution Language) - BPMN (Business Process Model and Notation)
